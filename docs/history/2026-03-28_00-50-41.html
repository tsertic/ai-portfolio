About Me

Engineer at the core.
Builder everywhere else.

I'm a full-stack developer at Arctis d.o.o. in Zagreb, where I build and maintain enterprise software for some serious organizations — Saudi Aramco, the European Space Agency, the European Court of Justice, University of Luxembourg, and others. The platform is Archibus IWMS, and my work spans Java backends, SQL optimization, frontend interfaces, and React integrations into systems that were never meant to run React.

I also do everything end-to-end: server setup, database configuration, Tomcat, deployment, documentation. The full lifecycle, every time.

Outside the day job, I build Next.js + TypeScript websites for clients — from initial design through deployment and ongoing maintenance. And I experiment: AI tools, Python scripts, 3D web, whatever problem is interesting that week.

When I'm not at the keyboard: basketball, cycling, hiking, diving, chess. Dad to Anja. Dog dad to Eevee.
