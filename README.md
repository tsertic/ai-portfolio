# AI-Regenerated Portfolio

A one-page portfolio that is **automatically redesigned every day** by an AI.
Static content (your name, projects, skills) stays the same — the design, layout, colors, and typography change daily.

Hosted for free on **GitHub Pages**, regenerated by **GitHub Actions**.

---

## How it works

```
data/content.json  →  scripts/generate.js  →  Claude / GPT-4 / Gemini  →  docs/index.html  →  GitHub Pages
                                ↑
                    runs daily via GitHub Actions cron
```

---

## Project structure

```
├── data/
│   └── content.json          ← your info (edit this)
├── scripts/
│   └── generate.js           ← AI generator script
├── docs/
│   └── index.html            ← generated portfolio (auto-updated daily)
├── .github/
│   └── workflows/
│       └── daily.yml         ← cron job (runs every day at 06:00 UTC)
├── .env.example              ← copy to .env for local use
└── package.json
```

## Setup guide

### 1. Fork / create the GitHub repo

1. Go to [github.com](https://github.com) → **New repository**
2. Name it e.g. `ai-portfolio` — make it **Public**
3. Clone it locally and copy these project files into it
4. Push to GitHub

### 2. Enable GitHub Pages

1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Branch: `main`, Folder: `/docs`
4. Click **Save**

Your site will be live at `https://YOUR_USERNAME.github.io/REPO_NAME/`

### 3. Add your API key as a GitHub Secret

1. Go to your repo → **Settings** → **Secrets and variables** → **Actions**
2. Click **New repository secret**
3. Add the secret for your chosen provider:

| Provider | Secret name | Where to get key |
|----------|-------------|------------------|
| Claude   | `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) |
| OpenAI   | `OPENAI_API_KEY` | [platform.openai.com](https://platform.openai.com) |
| Gemini   | `GEMINI_API_KEY` | [aistudio.google.com](https://aistudio.google.com) |

### 4. Set the provider (optional)

By default the workflow uses Claude. To change the provider:

1. Go to **Settings** → **Secrets and variables** → **Actions** → **Variables** tab
2. Add a new variable: `AI_PROVIDER` = `claude` | `openai` | `gemini`

### 5. Trigger the first generation

1. Go to your repo → **Actions** tab
2. Click **Daily Portfolio Regeneration**
3. Click **Run workflow** → choose provider → **Run workflow**
4. Wait ~30 seconds for it to complete
5. Your live site will update automatically

## Running locally

```bash
# 1. Copy and fill in your .env
cp .env.example .env
# Edit .env with your API key and preferred provider

# 2. Run the generator
node scripts/generate.js

# 3. Open the result
open docs/index.html
```

Or use the npm shortcuts:

```bash
npm run generate          # uses provider from .env
npm run generate:claude   # force Claude
npm run generate:openai   # force OpenAI
npm run generate:gemini   # force Gemini
```

---

## Customizing your content

Edit `data/content.json` — this is the only file you need to touch to update your personal info, projects, and skills. The AI will use it as-is every time it generates a new design.

---

## Changing the schedule

Edit `.github/workflows/daily.yml`, line:

```yaml
- cron: "0 6 * * *"   # every day at 06:00 UTC
```

Use [crontab.guru](https://crontab.guru) to build your preferred schedule.
